### 챌린지 자동화 프로그램 | 챌린지 캐치 ###

* 챌린지 출결 확인 자동화 프로그램 - 챌린지 캐치는 한 번의 클릭으로 챌린지 출결 데이터를 개인별, 날짜별로 자동 생성하며, 전체 정리된 데이터는 엑셀파일로 제공합니다.


### 챌린지 캐치 사용법 ###

1. 챌린지 이름, 챌린지 완료 여부를 알 수 있는 키워드를 입력하고, 카카오톡 대화 "열기" 버튼을 통해 텍스트 파일을 열어 주세요! (카카오톡에서 대화 내용 내보내기로 생성된 파일이 필요합니다.)

2. 텍스트 파일이 정상적으로 로드 되었다면, 파일 내용이 화면에 나타납니다. 이제 "시작" 버튼을 클릭하면 1초~3초 이내 데이터 분석과 파일 생성을 완료합니다.

3. 분석된 챌린지 출결 데이터는 "챌린지 이름"으로 만든 폴더 하위에 모두 저장합니다. (전체 정리 파일, 개인별 & 날짜별 출결 현황 파일)

4. 파일 저장과는 별도로 전체 현황을 새로운 윈도우 창을 통해 확인할 수 있습니다. 이때에도 특정 부분만 필터해 파일로 저장할 수 있습니다.

5. 프로그램 사용 중 에러가 발생하거나, 오작동 하더라도 PC 자체 또는 타 프로그램에 영향을 끼치지 않습니다. 예상치 못한 오류가 발생했을 경우, 프로그램 종료 후 다시 시작해 주세요.

6. 챌린지 캐치는 카카오톡 대화 내용을 기반으로 하는 텍스트 데이터 처리 프로그램입니다.
